# 🚀 CodeForage – Modern EdTech Learning Platform

CodeForage is a modern, responsive EdTech web application designed to provide an engaging online learning experience for aspiring developers. Built with **React**, **Vite**, and **Tailwind CSS**, the platform offers a clean and intuitive interface for exploring programming courses, learning paths, and career-focused resources.

## 🌐 Live Demo

**Website:** https://codeforageapp-odsiquib2-sirisandhila-s-projects.vercel.app/

---

## ✨ Features

* 📱 Fully responsive design
* ⚡ Fast performance with Vite
* 🎨 Modern UI built using Tailwind CSS
* 🧭 Interactive navigation
* 📚 Course showcase and learning paths
* 👨‍🏫 Mentor and instructor sections
* 💼 Placement statistics and success stories
* 💬 Student testimonials
* ❓ FAQ section
* 🌙 Clean and user-friendly interface
* ♻️ Reusable React components
* 🚀 Optimized for deployment on Vercel

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/<your-username>/<repository-name>.git
```

### Navigate to the Project

```bash
cd <repository-name>
```

### Install Dependencies

```bash
npm install
```

### Start the Development Server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
